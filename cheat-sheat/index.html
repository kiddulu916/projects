CSS Elements

Layout Properties

Tag Description
'display' The display property specifies the type of rendering box.
'position' The span element represents a section in a document
'float' The float property specifies how an element should float.
'clear' The clear property specifies if an element can be floated.
'width' The width property specifies the width of an element.
'height' The height property specifies the height of an element.
'margin' The margin property specifies the margin of an element.
'padding' The padding property specifies the padding of an element.
'box-sizing' The box-sizing property specifies how the total width and height of an element are calculated.

Tag Description

'text-align' The text-align property specifies the text alignment of an element.

'word-spacing' The word-spacing property specifies the word spacing of an element.

Color and Background Properties

Tag Description
'color' The color property specifies the color of text.
'background-color' The background-color property specifies the background color of an element.
'background-image' The background-image property specifies the background image of an element.
'background-repeat' The background-repeat property specifies whether the background image should be repeated.
'background-position' The background-position property specifies the position of the background image.
'background-attachment' The background-attachment property specifies whether the background image should be fixed or scroll.
'background-size' The background-size property specifies the size of the background image.
'opacity' The opacity property specifies the opacity of an element.

Flexbox Properties

Tag Description
'display' The display property specifies the type of rendering box.
'flex-direction' The flex-direction property specifies the direction of the flexible items.
'flex-wrap' The flex-wrap property specifies whether the flexible items should wrap or not.
'justify-content' The justify-content property specifies how the flexible items should be positioned in the flex container.
'align-items' The align-items property specifies how the flexible items should be aligned in the flex container.
'align-content' The align-content property specifies how the flexible items should be aligned in the flex container when there is extra space in the cross-axis.

Grid Properties

Tag Description
'grid-template-columns' The grid-template-columns property specifies the number of columns in the grid.
'grid-template-rows' The grid-template-rows property specifies the number of rows in the grid.
'grid-column' The grid-column property specifies the number of columns an element should span.
'grid-row' The grid-row property specifies the number of rows an element should span.

Transition & Animation Properties

Tag Description
'transition' The transition property specifies the animation effect.
'animation' The animation property specifies the animation effect.

Other Properties

Tag Description
'cursor' The cursor property specifies the mouse cursor to be displayed when the mouse pointer is over an element.
'overflow' The overflow property specifies what should happen if content overflows an element's box.
'z-index' The z-index property specifies the stack order of an element.
